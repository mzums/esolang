Welcome to GardenersScript

GardenersScript is an esoteric programming language (esolang) where you manipulate "garden plots" as variables. The language uses gardening metaphors for programming concepts, making it both educational and fun.

Key Features

Intuitive Metaphors

Plant seeds (variables), water them (add values), prune (subtract), and harvest (output).

Two Data Types

Supports integers and strings for flexible programming.

Control Structures

Loops (bloom/wither) and conditionals (check) for complex programs.

Functions

Create reusable code blocks with parameters and return values.

Getting Started

To start using GardenersScript:

  1. Read the documentation to learn the syntax
  2. Write your first program (e.g., Hello World)
  3. Run it using the GardenersScript interpreter

Example Program

# Hello World in GardenersScript
plant msg "Hello, World!"
tend msg
harvest msg
